# MediaWiki Installation

MediaWiki was installed on the Ubuntu server to provide a wiki-based knowledge base. The installation involved downloading the MediaWiki package, extracting it into the Apache web root directory, configuring the database, completing the web-based installation wizard, and and uploading the generated `LocalSettings.php` configuration file.

---

## Step 1 – Download the MediaWiki Package

Downloaded the MediaWiki package onto the Ubuntu server.

![SS1 – Downloading the MediaWiki Package](images/SS1%20%E2%80%93%20Downloading%20the%20MediaWiki%20Package.png)

---

## Step 2 – Extract the Package

Extracted the MediaWiki package into `/var/www/html/mediawiki` and configured the appropriate ownership and permissions.

![SS2 – MediaWiki Extracted into the Web Root Directory](images/SS2%20%E2%80%93%20MediaWiki%20Extracted%20into%20the%20Web%20Root%20Directory.png)

---

## Step 3 – Run the Installation Wizard

Accessed the MediaWiki installation wizard through the browser and completed the initial configuration.

![SS3 – MediaWiki Installation Wizard Ready](images/SS3%20%E2%80%93%20MediaWiki%20Installation%20Wizard%20Ready.png)

---

## Step 4 – Complete the Installation

Configured the wiki name, database connection, and administrator account. The generated `LocalSettings.php` file was uploaded to complete the installation.

![SS4 – MediaWiki Installation Completed Successfully](images/SS4%20%E2%80%93%20MediaWiki%20Installation%20Completed%20Successfully.png)

---

## Step 5 – Verify the Installation

Verified that the **Staff Knowledge Base** was successfully installed and accessible through the configured HTTPS domain.

![SS5 – MediaWiki Successfully Installed](images/SS5%20%E2%80%93%20MediaWiki%20Successfully%20Installed.png)
